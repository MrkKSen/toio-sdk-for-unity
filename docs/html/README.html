チュートリアル

基本

  • シーン作成
  • 移動する
  • 音を鳴らす
  • LED を発光する
  • イベントを検知(ボタン, 傾き, 衝突, 座標と角度)
  • 複数のキューブを動かす
  • ソースコード簡略化
  • 途中接続/途中切断

便利な移動機能 - CubeHandle

  • CubeManager を使ったキューブの同期制御
  • CubeHandle
    • CubeHandle の Move 関数と MoveRaw 関数
    • キューブとの通信量を抑える One-shot メソッド
    • 指定した座標/方向に到達する Closed-Loop メソッド
  • Follow TargetPole デモ

集団制御 - Navigator

  • CubeNavigator
    • CubeManager を使って CubeNavigator を利用する
      • 非同期でキューブを制御する場合
      • 同期でキューブを制御する場合
      • CubeManager を使わないで CubeNavigator を利用する
    • CubeNavigator による衝突回避
      • 衝突を回避しつつ目標に移動する Navi2Target 関数
      • 目標から離れる NaviAwayTarget 関数
    • ボイドによる集団制御
    • ボイド + 衝突回避

サンプル

基礎

  • Sample_UI

    ユーザーが UI を操作して直接キューブを動かすサンプルです。

  • Sample_Circling

    CubeNavigator のモードの違いによって、多数台のキューブの挙動がどのように変わるか確認するサンプルです。

  • Sample_Cross

    CubeNavigator を使って多台数のキューブが衝突回避しながら移動をするサンプルです。

応用

  • Sample_MultiMat

    複数のマットを併せて一枚の大きいマットとして使うサンプルです。

  • Sample_Bluetooth

    低レベルモジュールである BLE インタフェースを直接利用してキューブと通信するサンプルです。

  • Sample_WebGL

    ブラウザで動作するウェブアプリケーションのサンプル集です。

留意事項

シミュレータの効果音の再生機能は実装されていますが、効果音自体は実装されていません。 現在は一つだけのダミー効果音が入っています。

効果音を実装するには、CubeSimulator クラスの _InitPresetSounds 関数の中で効果音を定義します。

// Sound Preset を設定
private void _InitPresetSounds(){
    Cube.SoundOperation[] sounds = new Cube.SoundOperation[3];
    sounds[0] = new Cube.SoundOperation(200, 255, 48);
    sounds[1] = new Cube.SoundOperation(200, 255, 50);
    sounds[2] = new Cube.SoundOperation(200, 255, 52);
    this.presetSounds.Add(sounds);
}

Cube.SoundOperation の説明は Cube の使い方 を参照してください。
